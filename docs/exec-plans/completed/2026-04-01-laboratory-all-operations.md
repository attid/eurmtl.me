# Laboratory All Operations

## Контекст
В `Laboratory` операции сейчас разнесены по тематическим вкладкам, что удобно для частых сценариев (например, платежей), но затрудняет поиск редких операций. Нужна отдельная вкладка `All`, не меняющая текущий default UX, и фильтр по названию операции внутри этой вкладки.

## План
1. [x] Добавить регресс-тесты на HTML `/lab`: вкладка `All`, поле фильтра и наличие в полном списке как платежной, так и редкой операции.
2. [x] Запустить таргетные тесты и убедиться, что новые ожидания падают до правок.
3. [x] Минимально обновить `templates/tabler_laboratory.html`: добавить вкладку `All`, общий список операций и JS-фильтр по названию.
4. [x] Повторно запустить таргетные тесты.
5. [x] Прогнать `just check-changed`.

## Результат
- Добавлена новая вкладка `All` без изменения default-вкладки.
- Во вкладке `All` добавлен полный список операций и текстовый фильтр по названию.
- Добавлен роутерный регресс-тест на наличие вкладки, фильтра и операций в полном списке.
