# LLM TXT Agent Entry

## Контекст
Сайту нужен машинный вход `GET /llms.txt`, чтобы агент без браузера мог быстро понять, какие HTTP-эндпоинты доступны, какие форматы данных ожидаются и какие операции стоит использовать вместо HTML-интерфейса.

## План изменений

2. [ ] Добавить в `routers/index.py` минимальный plain-text роут `GET /llms.txt`.
3. [ ] Описать в `llm.txt` основные машинные маршруты сайта и короткие правила работы без браузера.
4. [ ] Прогнать релевантные тесты и `check-changed`.
